# Shop app

Верстка мобильно-ориентированная по бесплатному макету для портфолио.

### Информация

*Ссылка для просмотра*: <https://natalyaberdnikova.github.io/ShopApp/>

### Установка

Чтобы установить проект локально, необходимо выполнить 3 простых шага:

1. `git clone <repo> .`
2. `npm install`
3. `gulp`
